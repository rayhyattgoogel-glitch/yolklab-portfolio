# This is NOT the Next.js you know

Next.js 16 has breaking changes from 15.x — APIs, conventions, and file structure may differ from your training data. Read the relevant guide in `node_modules/next/dist/docs/` before writing any code. In particular: page `params` are Promises, `output: "export"` forbids ISR / Edge / server actions, and Tailwind v4 has no `tailwind.config.ts` (config lives in CSS).
